# Unciv鳇脯菌校：全局篇

> 作者：遗封
>
> 完稿日期：2025年2月13日

序言：本作是为了接下来需要参与未来夏季赛以及为提升pvp水平的萌新提供帮助而所创作的教程，希望各位在初步理解和掌握本篇芝士后能够得到足够的提升。

## 第一节：全局时段规划

首先，我们要先了解在pvp中，全局的时段规划，也就是什么时间段该做什么样的事情，而不是南辕北辙地做一些匪夷所思的规划，从而给自己带来巨大压力最终导致失败。

由此我们可以把全局分为大致的四个时段：1.起始阶段（0\~25t）；2.发育阶段（26t\~40t）；3.对抗阶段（41t\~70t）；4.僵持阶段（70t+）（这里以极小为主进行规划划分，不同情况特殊对待）

1.  起始阶段（0\~25t）：

    这个阶段为玩家刚开始一穷二白的时期，在这一阶段玩家的初始可利用资源有：首都及其地块、有限的快乐数值和一个勇士单位。那么我们需要做的是：1.扩大己方控制地块和掌握更多发育资源；2.初步了解敌方信息，以及与对方进行该阶段的对抗，从而获取初步的正向对抗收益（包括但不限于初步压缩对方可控区间、击杀单位以及欺骗性策略误导对方决策）3.初步提高内政水平。

    那么这一阶段我们作为玩家在此目标指导下，要做的就是建造分城（对应目标1）、利用单位初步压制对方（对应目标2），建造内政建筑（如粮仓，纪念碑，神社）或利用工人改良（对应目标3），由于前期的产能较为有限，每种选择的产能投入方式【及初始建造军事单位开局（单位开）、纪念碑开局（碑开）、工人开局（工开）】与政策选择【传统、自主、荣誉】以及科技选择【制陶、家畜驯养、弓箭、采矿】，所侧重的目标比重都会出现不同，演化为不同的后续发展状况。

    举例为黑暗决斗打法之一的自主双弓打法：即建造顺序一般为纪念碑开局-弓箭-弓箭-移民，科技以弓箭起手，政策选择自主就是侧重于投入至目标1、2的决策，同时利用自主特性补足目标3需求。

    所以理解了自己在这一阶段所能达到的目标之后，再做决策和发展便清楚许多了。即在尽可能达到目标1、2、3的同时，也要注意敌方的动向，以及大致判断对方企图之后需要立刻做出反制措施，因为pvp里，玩家是游戏的主导者，一如下象棋二者互相博弈。

    例如你面对对方的压制性打法，对手出了单位压制，此时你正在起移民时，若对手的打法会影响你的移民时，你就需要做出反制手段了，即出单位进行对应反制的同时完成自己的战略部署。

    此外，我们需要记住的是，被速杀的可能仍然是有的，即遭受对手极端偏向目标2的打法所导致的大量损失，如新造移民和单位被俘虏杀害，或是单城自爆打法速通自己家。我们既要防止这种情况，也要减少自己使用这种极端性打法的想法，过度偏向某一目标进行运营，是具有极高风险的，容易适得其反。

2.  发育阶段（26t-40t）

    在稳步进入该阶段时，此时你的可利用资源有首都及其地块与分城及其地块，余下的快 乐数值，少量的可操作军事单位。那么这一阶段，你所要做的目标为：1.承载先前基础 上扩大己方控制地块和掌握更多发育资源；2.进一步了解敌方信息，以及与对方进行该 阶段的对抗，从而获取一定的正向对抗收益；3.提高内政水平，以加快己方成型。

    那么围绕这些目标，在这一阶段，我们所要做的事情算是非常多的，因为这一时期是双 方加快发育且伺机而动的时期，我们既要考虑是否进行内政建筑的进一步补充（在原先 基础上补充其他内政建筑）或是增加更多分城加强地形控制，同时也要加快己方军队的 成型速度，服务于对抗可能存在的偷袭，以及面对下一个阶段必然会爆发的激烈对抗。

    举例为我选择了纪念碑开局的自主打法之后，在这一时期，我需要加快自主政策的成型， 所以我选择了分城补充了纪念碑，同时利用工人加快改良地块设施，更快的为自己提供

    足够的内政收益，同时扩大我的军队，建造大量质量较高的军事单位预备下一步的对抗 （如布置己方单位防线或进攻布局），或是进行灵活的压制争取影响对方的成型速度。 从而获取足够的战略优势甚至是提前的胜利。

    当然，以防对手的偷袭也是至关重要的，记得多偷窥胜利进度中的排名与图表，细细察 看对手信息，从而做出最好的判断。

3.  对抗阶段（41t-70t）及僵持阶段（70t+）

    这一时期，我将其称之为对抗时期。可能有人会问，为什么这一时期才被称为对抗期？ 回答是：之前的对抗大多只是，微小的、局部的。一般情况下它很难迅速决定游戏胜负 但是，这一时期，是双方进入单位补给近乎饱和（近似理解为星际和帝国的满人口）， 内政发展较为充分，军队大体成型，控制区域需求再次扩大与地块资源随玩家控制而逐 渐紧张的时期。故而在此阶段的一般情况，大型且激烈的决战是必然会产生的，故而我 们所要做的便是想办法赢得决战的胜利，击溃敌方的主力部队抑或是夺取对方关键城市。

    至于一旦发生长时间难以改变对局的局面，也就是即使在对抗期也无法结束游戏的情况 下且进入了更漫长的70t后的僵持阶段。那么你所能做的，便是保障战线不崩溃的同时 通过大量的奇观和内政建筑与更高质量的部队来找出转机结束游戏，这必然是一个很漫 长的环节，还请玩家耐心面对。
